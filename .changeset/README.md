# Changesets

This folder is managed by [Changesets](https://github.com/changesets/changesets).

Run `npm run changeset` to record a change and its version bump. On push to
`main`, the Release workflow turns accumulated changesets into a "Version
Packages" PR; merging it publishes to npm.

See the [common questions](https://github.com/changesets/changesets/blob/main/docs/common-questions.md)
for more detail.
